Core Library

Kjøre Dart

  • Kompilerer til Javascript, kan kjøres i Chrome, Safari 5+ og Firefox 4+. Med optimalisering ender det på ca 28 KB.
  • VM på server

Pluss

  • Raskt å komme i gang med
  • Valgfri typing
  • Tilgang til DOM-treet
  • Isolate, Promise, SendPort/ReceivePort (concurrency)
  • OK standard-bibliotek (men ikke noe særlig som ikke et JavaScript-bibliotek kan gi deg)

Minus 

  • null
  • "Kjedelig"
  • Ingen native browserstøtte (enda)
  • CoffeeScript > Dart
  • Kompilerer til ganske håpløs JavaScript-kode (http://webreflection.blogspot.com/2011/10/what-is-wrong-about-17259-lines-of-code.html)
